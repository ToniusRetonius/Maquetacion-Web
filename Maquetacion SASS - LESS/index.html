Nuevos elementos del standard

con el surgimiento del nuevo standard, se incorporaron nuevas etiquetas que facilitarion a los desarrolladores la implementacion de estadisticas, contenido multimedia e incluso API's de dibujo. Si bien la mayoria de los navegadores actuales soporta dichas Caracteristicas, debemos tener en cuenta que todavia existen algunos que no y por ende, nuestro dise;o debe contemplar dichas situaciones. Por eso podemos consultar en sitios como CAN I USE donde nos mostrara una lista dde compatibilidad de elementos.

Elementos estadisticos

Cada vez que necesitemos comparar valores o evaluar una situacion especifica podemos utilizar 2 etiquetas: <progress que debe implementarse cuando estamos llevando a cabo una tarea que aun no fue finalizada, ya que se utiliza para medir el avance dentro de la misma, ej: el progreso de la capacitacion <meter que debe implementarse cuando la tarea fue finalizada y debemos establecer una comparacion de valores, ej: el resultado de un examen

API's de dibujo

Actualmente disponemos de dos herramientas para dibujar dentro de nuestro HTML, por un lado < SVG (Scalable Vector Graphics) que utiliza como el lenguaje base XML y permite la creacion de graficos vectoriales mediante la implementacion de lineas y curvas. Por otro lado, existe CANVAS (lienzo) que utiliza como lenguaje base JavaScript y permite la creacion de graficos pixelares o tambien conocidos como mapa de bits. Si bien ambas herramientas nos permiten realizar graficos dentro de nuestro sitio, tambien podemos editar graficos existentes.

preprocesadores CSS

Pseudolenguajes de CSS

Los Pseudolenguajes son herramientas que utilizan una sintaxis particular. Toman como base el lenguaje de hojas de estilos. Permiten la implementacion de todos los metodos y Caracteristicas que existen en CSS, pero ademas agrega algunas funcionalidades extra; para utilizarlos debemos crear un archivo con el formato del preprocesador que vayamos a utilizar y luego generamos la hoja de estilos resultante mediante el uso de un compilador, es decir, un interprete de las instrucciones previamente escritas en el codigo que las convierte a un archivo CSS convencional.

LESS (Leaner Style Sheets)

Preprocesador que utiliza JavaScript como lenguaje compilador. Tambien implementa varias caracteristicas del lenguaje como la asincronicidad. Para instalarlo debemos utilizar el siguiente comando con NodeJs npm install -g (global) less y una vez creada la hoja de estilos la compilamos con el siguiente comando lessc 'input.less' 'output.less'

Lamentablemente el preprocesado no posee un metodo de observacion, pero para ello podemos instalar otro paquete llamado Less Watch Compiler utilizando el comando con NodeJs npm install -g less-watch-compiler y para compilar y observar los cambios para actualizar las hojas de estilo utilizamos el comando less-watch-compiler 'input/' 'output/' 'input.less'

SASS (Syntactically Awesome Style Sheets)

Preprocesador que utiliza RUBY como lemguaje compilador. A diferencia de LESS, todos sus metodos deben estar declarado en orden; ya que de lo contrario arrojara un error de compilacion, pero posee algunas caracteristicas automatizadas: como la generacion de bucles de distinto tipo o la distincion de metodos.

Para su instalacion debemos ejecutar el comando de NodeJs npm install -g sass y para compilar nuestros archivos de manera automatizada simplemente debemos ingresar el comando sass --watch input.scss, cabe destacar que este preprocesador posee dos tipos de sintaxis : SASS (formato original - sin scope) y SCSS (revision con implementacion de signos)

Efectos Visuales

Disenando con CSS

Actualmente, CSS nos permite utilizar reglas cuyo funcionamiento es similar al de los programas de diseno de Adobe como Photoshop, Illustrator o Animate; pero utilizando solamente codigo del lenguaje. Esto se debe a que toma la sintaxis que genera dichos programas para generar las reglas pertinentes. A diferencia de los atributos clasicos, estos no solamente afectan a la etiqueta seleccionada, sino que tambien afectan su contenido.

Transformaciones

Cada etiqueta posee puntos de ajuste quepueden ser modificados gracias al atributos transform, el cual permite entre otras cosas, cambiar las dimensiones, desplazar la etiqueta, rotarla o sesgarla, sin afectar el flow. Podemos combinar estos efectos con otro atributo conocido como transition que permite generar los estados intermedios cuando se realiza un cambio en determinados atributos, como por ejemplo un :hover.
